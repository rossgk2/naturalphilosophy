# Logic

I was very unsettled to learn that there are multiple different logical systems we can use to understand the universe. For instance...

* In *classical logic*, we assume that there are only two truth values, true and false, and have not(true) = false and not(false) = true. Every proposition in classical logic is either true or false.
* In *constructivist logic*, we assume that there are three truth values, true, false, and indeterminate, and have not(true) = {false, indeterminate} and not(false) = {true, indeterminate}. The truth value of a proposition in constructivist logic is indeterminate until it has been explicitly determined. (Constructivist logic is very useful in computer science because every true constructivist theorem corresponds to an algorithm that can explicitly construct the result.)

Finding this out was like learning that the fundamental logic underpinning the universe, which I thought to be whole, was actually fragmented... all this time.

I couldn't accept this. So I thought about it, and realized that the difference between the two systems can be stated in a pleasing, philosophical way: 

* In classical logic, the universe exists whether you are there to discover it or not.
* In constructivist logic, the only parts of the logical universe that exist are those that you have uncovered.

So perhaps a better way to see things is that these logical systems aren't the fundamental, universal logic underpinning all things we thought they were. They are just different views onto that universal logic, illuminating different aspects of it.
